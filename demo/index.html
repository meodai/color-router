Token Examples

Try these token syntax examples in the color definition field above:

Color Tokens

  • val.color.hex('#3498db')
  • val.color.lch('65% 40 250')
  • val.color.oklch('0.7 0.15 250')
  • val.color.css('rgba(52, 152, 219, 0.8)')

Size & Space Tokens

  • val.size.rem(2.5)
  • val.size.px(24)
  • val.space.rem(1)
  • val.space.px(16)

Font Tokens

  • val.font.family('Inter, sans-serif')
  • val.font.size('2.5rem')
  • val.font.weight('700')
  • val.font.style('italic')

Reference Tokens

  • ref('base.primary')
  • ref('ramp.500')
  • ref('surface.background')

Function Examples

  • func.color.bestContrastWith('base.primary', 'ramp')
  • func.color.mix('base.accent', 'base.light', 0.3, 'oklab')
  • func.color.lighten('demo.primary', 0.2)
  • func.color.relativeTo('base.accent', 'r g b / 0.7')
